# Credit-Card-Dashboards

## Project Objective
To develop a **comprehensive credit card weekly dashboard** that provides **real-time insights** into key performance metrics and trends, enabling stakeholders to effectively **monitor and analyze credit card operations**.

---

## Project Overview
This project focuses on building an **interactive Power BI dashboard** using **transaction and customer data stored in a SQL database**. The dashboard delivers both **weekly performance tracking** and **year-to-date (YTD) insights** to support data-driven decision-making.

---

## Key Insights – Week 53 (31st Dec)

### Week-over-Week (WoW) Changes
- **Revenue** increased by **28.8%**
---

## Year-to-Date (YTD) Overview
- **Overall Revenue:** 57M  
- **Total Interest Earned:** 8M  
- **Total Transaction Amount:** 46M  
- **Revenue by Gender:**
  - Male customers: **31M**
  - Female customers: **26M**
- **Card Type Contribution:**
  - Blue & Silver cards contribute **93%** of overall transactions
- **Top Contributing States:**
  - TX, NY, and CA contribute **68%** of total revenue
- **Overall Activation Rate:** 57.5%  
- **Overall Delinquent Rate:** 6.06%

---

## Dashboard Features
- Weekly and YTD performance tracking
- Revenue and transaction analysis
- Customer segmentation by gender and geography
- Credit card type contribution analysis
- Activation and delinquency rate monitoring
- Interactive filters for deeper insights

---

## Tools & Technologies
- **Power BI** – Dashboard development & visualization  
- **SQL** – Data extraction and transformation  
- **Relational Database** – Source of transaction and customer data  

---

## Actions & Implementation
- Developed an **interactive Power BI dashboard** integrating customer and transaction data from SQL.
- Streamlined **data processing and analysis** to monitor KPIs and trends efficiently.
- Delivered **actionable insights** to stakeholders to support strategic and operational decisions.

---

## Business Impact
- Enabled real-time monitoring of credit card performance
- Improved visibility into customer behaviour and revenue drivers
- Supported faster and more informed decision-making for stakeholders
